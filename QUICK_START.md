# 快速开始指南

## 项目概述

股票龙虎榜数据展示应用，提供实时和区间数据查询功能。

## 环境要求

- Node.js 18+
- npm 或 yarn

## 安装和运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:3000` 启动

### 3. 构建生产版本

```bash
npm run build
npm start
```

## 主要功能

### 📊 实时数据
- 显示最新的龙虎榜数据
- 自动刷新（交易时间内每 6 秒）
- 手动刷新按钮
- 分页支持

### 📈 区间数据
- 选择时间范围查询数据
- 支持 5 分钟最小跨度
- 交易时间：9:25-11:30, 13:05-15:00
- 分页支持

### ⚙️ 列管理
- 显示/隐藏列
- 拖拽调整列顺序
- 重置为默认配置

## 使用说明

### 查看实时数据

1. 打开应用
2. 在"实时数据"卡片中查看最新数据
3. 点击刷新按钮手动更新
4. 使用分页查看更多数据

### 查看区间数据

1. 滚动到"区间数据"卡片
2. 使用滑块选择时间范围
3. 数据会自动加载
4. 使用分页查看更多数据

### 自定义列显示

1. 点击"列设置"按钮（齿轮图标）
2. 在对话框中：
   - 勾选/取消勾选列以显示/隐藏
   - 拖拽列名调整顺序
3. 点击"重置"恢复默认配置
4. 点击"关闭"保存设置

## 常见问题

### Q: 为什么看不到数据？

**A:** 可能的原因：
1. 网络连接问题 - 检查网络
2. 非交易时间 - 只在工作日 9:25-11:30 或 13:05-15:00 有数据
3. API 服务问题 - 稍后重试

### Q: 自动刷新不工作？

**A:** 自动刷新仅在交易时间启用。检查：
1. 当前时间是否在交易时间内
2. 是否为工作日（周一至周五）

### Q: 如何导出数据？

**A:** 当前版本不支持导出。可以：
1. 使用浏览器的截图功能
2. 使用开发者工具复制表格数据

### Q: 列设置会保存吗？

**A:** 当前版本使用内存存储，刷新页面后会重置。
